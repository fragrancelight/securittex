Introduction

Details

For Tradexpro Exchange we made a feature named Blog-News. It will run with the help of Tradexpro Exchange (Main Project). It typically includes features such as the ability to write and publish Blog and News, organize and categorize posts, and include multimedia elements such as images and videos. Users can also typically interact with the app by commenting on and sharing posts, as well as customizing the design and layout of their Blog and News.

Getting Started

The application has a few system requirements, so you will need to make sure your server meets the following requirements:

Admin side requirement :

  1. It will run with Tradexpro Exchange Main Project, So everything is same as main project

User side requirement :

  1. Nodejs 16.16.0
  2. Yarn 1.22.19 or above
  3. Nginx

Installation Process

Addon Installation

Addon Name: BlogNews

  • It's a super easy installation process

    • Warning ! You must have the latest updated admin file (tradexpro laravel), then start the next process

    Remember : It's just an addon. It does not run alone. It will run with the help of "Tradexpro Exchange" project.

    So you must have the "Tradexpro Exchange" Project first

    Here we integrate the new addon "Tradexpro Blog & News" to Tradexpro Exchange project that was already running

  • Fresh tradexpro project that is running already

  • Look here is "Addons" tab

  • Click on "Addons List" from "Addons" tab

  • This is the current folder structure

    • To integrate this addon, we must follow the bellow procedure

    # Step 1 :


    Go to project directory and open the terminal or Open the terminal then go the project directory

  • Run this command in project directory

    "composer install --ignore-platform-reqs"

    "composer dumpa --ignore-platform-reqs"

    • # Step 2 :

    First create "Modules" folder in project root directroy (if not exist) and go in.


    # Step 3 :

    Put the zip file that we provide in the modules folder.

    Here extract zip file that you already put in modules folder.

    Now delete the zip file from modules folder.

    # Step 4 :

    Then go to the project path (like ..tradexpro-exchange/) , and run some commands below.

    "php artisan module:enable BlogNews"


  • Then you get this message after enable the module successfully.

    • # Step 5 :

  • Then Need to run the module migration command

    • "php artisan module:migrate BlogNews"


  • Then you get this after successfully run the command

  • Then Need to run the module seeder command

    • "php artisan module:seed BlogNews"

  • Affter runnig this command you will get response like this.

    • Installation is done

    # Step 6

    Click on "Addons Settings" from "Addons" tab

    Now you can enable/disable this module from Addons Settings

  • Click on "Addons List" from "Addons" tab
